# WaTching

> Spring Boot를 활용해 제작한 영화예매 사이트 WaTching 입니다.<br>

### ✔️개발 동기 및 개요
- "WaTching"은 실무에 필요한 웹 개발 기술을 학습하고, 사용자 친화적인 영화 예매 시스템을 구현해보고자 시작한 프로젝트입니다. 


### ✔️ 기간
  `2024.08.30 ~ 2024.09.04 (4일 소요)`
